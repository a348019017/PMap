
## 依赖
UI采用Element或HTML，并采用Vue组件进行集成



## 安装和使用
```````````````````````````````````
cnpm install @pmap-cesium/vue2 --save
```````````````````````````````````

```````````````````````````````````
import {VueCesiumShutter} from "@pmap-cesium/vue2"
import "@pmap-cesium/vue2/dist/style.css"

<template>
   <Map></Map>
   <VueCesiumShutter></VueCesiumShutter>
</template>
````````````````````````````````````

## Content

* VueCesiumShutter
  封装的一个卷帘组件，未来支持横向和竖行
