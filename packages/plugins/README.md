## 简介
Cesium的一些中低粒度常用封装类，如编辑模型等操作。一般不包含UI操作，需要UI部分请查看@pmap-cesium/vueCom的组件封装

## ChangeLog
1.0.0 更新三个插件类


## Contents

* CesiumShutter

   卷帘基础封装暂不含UI

* CesiumModelEditor

  模型编辑类不含UI

* CesiumModelEditor

  模型编辑类，带操作轴
